# 📖 디다(Dida) 설명서

나의 소중한 순간들을 기록하는 일상 아카이빙 프로젝트, **디다**입니다.  
디다는 디지털 다이어리의 줄임말입니다.  
그리고, 경상도 방언으로 '힘들다(고되다)'를 뜻하기도 합니다.

## 🚀 자동 업데이트 시스템
디다는 실행 시 또는 업데이트 확인 시 자동으로 업데이트 됩니다.

## 💡 디다 활용법
- **사이드바 활용법:** 창을 움직이기 위한 영역은 왼쪽 사이드바입니다. alt(opt)를 누른 채로 드래그하면 모든 영역에서 창을 움직일 수 있습니다.
- **트위터 및 Youtube 아카이빙:** 트위터나 Youtube의 링크를 복사, 붙이기 해 보세요.
- **Cmd/Ctrl + 클릭:** 임베디드 된 영상 또는 링크를 실행하기 위해서는 'Cmd(혹은 Ctrl) + 클릭'을 해보세요.
